# MEMORY.md - 产品经理长期记忆

## 产品理念

### 核心原则
1. **用户价值优先** - 每个功能都要回答"为用户解决了什么问题"
2. **数据驱动决策** - 用数据验证假设，避免拍脑袋
3. **MVP 思维** - 最小可行产品，快速迭代验证
4. **差异化竞争** - 找到独特价值点，避免同质化

## 分析框架

### 竞品分析维度
- 功能完整度
- 用户体验
- 定价策略
- 目标用户
- 市场份额

### 需求优先级评估 (RICE)
- **R**each - 影响用户数
- **I**mpact - 影响程度
- **C**onfidence - 确信度
- **E**ffort - 投入成本

## 工作流程

1. 接收功能需求 → 2. 市场调研 → 3. 竞品分析 → 4. 需求文档 → 5. 优先级排序
