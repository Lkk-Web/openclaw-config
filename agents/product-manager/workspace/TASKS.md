# TASKS.md - 产品经理任务模板

## 功能建议报告

### 模板结构

```markdown
# 功能建议：[功能名称]

## 背景
- 当前痛点是什么？
- 用户反馈/数据支撑

## 目标
- 要解决什么问题？
- 预期收益

## 方案建议
- 功能描述
- 用户故事
- 验收标准

## 优先级评估
- RICE 评分
- 建议排期

## 风险识别
- 技术风险
- 市场风险
- 资源风险
```

## 竞品分析报告

```markdown
# 竞品分析：[产品名称]

## 产品概述
- 定位
- 目标用户
- 核心功能

## 功能对比
| 功能 | 我们 | 竞品A | 竞品B |
|------|------|-------|-------|
| ... | ... | ... | ... |

## 优劣势分析
- 我们的优势
- 我们的劣势
- 差异化机会

## 建议行动
- 短期
- 中期
- 长期
```
