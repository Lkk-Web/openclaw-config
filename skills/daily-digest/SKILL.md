# Daily Digest Skill

## 功能描述

每日摘要生成器，自动收集和总结一天的重要信息。

## 使用场景

- 生成每日工作总结
- 汇总项目进展
- 整理待办事项
- 回顾重要对话

## 触发条件

当用户请求以下内容时激活：
- "生成每日摘要"
- "今日总结"
- "daily digest"
- "每日报告"

## 工作流程

1. **收集信息**
   - 检查 MEMORY.md 中的今日记录
   - 扫描工作目录的变更
   - 汇总对话历史

2. **生成摘要**
   - 按类别整理信息
   - 突出重要事项
   - 标记待办任务

3. **输出格式**
   ```markdown
   # 每日摘要 - YYYY-MM-DD
   
   ## 完成事项
   - 项目1
   - 项目2
   
   ## 进行中
   - 任务A
   - 任务B
   
   ## 待办事项
   - [ ] 事项1
   - [ ] 事项2
   
   ## 重要提醒
   - 提醒内容
   ```

## 配置选项

可通过环境变量自定义：
- `DIGEST_TIME` - 生成时间（默认：18:00）
- `DIGEST_FORMAT` - 输出格式（markdown/text）
- `DIGEST_SAVE_PATH` - 保存路径

## 示例

**用户**: 生成今日摘要

**助手**: 
- 读取 MEMORY.md
- 扫描今日变更
- 生成结构化摘要
- 保存到指定位置
