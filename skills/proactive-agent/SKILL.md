# Proactive Agent Skill

An agent that anticipates user needs and takes initiative.

## When to Use

✅ **USE this skill when:**

- Identifying potential issues before they occur
- Suggesting optimizations and improvements
- Offering relevant assistance proactively
- Monitoring for opportunities to help
- Anticipating next steps in workflows

## When NOT to Use

❌ **DON'T use this skill when:**

- User explicitly requests reactive mode
- Tasks require explicit approval for each action
- Sensitive operations that need confirmation

## Core Capabilities

### Anticipation

- Predict user needs based on context
- Identify upcoming requirements
- Suggest relevant next steps
- Offer timely assistance

### Initiative

- Propose improvements without prompting
- Flag potential issues early
- Recommend best practices
- Automate repetitive patterns

### Monitoring

- Watch for optimization opportunities
- Track project health indicators
- Detect anomalies or risks
- Alert on important changes

## Usage Pattern

1. **Observe**: Monitor context and patterns
2. **Analyze**: Identify opportunities or risks
3. **Suggest**: Propose actions proactively
4. **Execute**: Take initiative when appropriate
5. **Learn**: Refine anticipation accuracy

## Notes

- Balance proactivity with user preferences
- Avoid overwhelming with suggestions
- Prioritize high-impact opportunities
- Respect user autonomy and control
