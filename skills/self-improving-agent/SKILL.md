# Self-Improving Agent Skill

An agent that continuously learns and improves from its interactions.

## When to Use

✅ **USE this skill when:**

- Analyzing agent performance metrics
- Identifying patterns in successful/failed tasks
- Updating decision-making strategies
- Optimizing workflow efficiency
- Learning from user feedback

## When NOT to Use

❌ **DON'T use this skill when:**

- Simple one-off tasks
- Tasks requiring immediate execution without analysis
- Static rule-based operations

## Core Capabilities

### Performance Analysis

Analyze past interactions to identify:
- Success patterns
- Common failure points
- Efficiency bottlenecks
- User preference trends

### Strategy Optimization

Update agent behavior based on:
- Task completion rates
- User satisfaction signals
- Resource utilization
- Response quality metrics

### Continuous Learning

- Store learned patterns in memory
- Apply insights to future tasks
- Adapt to changing requirements
- Refine communication style

## Usage Pattern

1. **Collect Data**: Gather interaction history and outcomes
2. **Analyze**: Identify patterns and insights
3. **Update**: Modify strategies and behaviors
4. **Validate**: Test improvements in real scenarios
5. **Iterate**: Continuously refine based on results

## Notes

- Requires access to interaction history
- Updates should be incremental and validated
- Balance exploration vs exploitation
- Maintain transparency about changes
